# SHA-256

**类别**: 密码学

## 概述

SHA-256 是 SHA-2 家族中的加密哈希函数，生成 256 位（32 字节）的哈希值。

## 原理

### 算法结构

SHA-256 是 Merkle-Damgård 结构的哈希函数，处理 512 位（64 字节）的消息块。

**处理流程**：
1. **消息填充**: 添加 1 位，然后填充零，最后添加 64 位的消息长度
2. **初始化寄存器**: 8 个 32 位寄存器（A, B, C, D, E, F, G, H）
3. **处理每个块**: 对每个 512 位块进行 64 轮处理
4. **输出**: 256 位的哈希值

### 安全性

- **抗碰撞性**: 理论上需要 2^128 次操作才能找到碰撞
- **抗原像性**: 理论上需要 2^256 次操作才能找到原像
- **目前没有已知的实际攻击**
