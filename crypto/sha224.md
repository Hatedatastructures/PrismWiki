# sha224.hpp

> 源码: `include/prism/crypto/sha224.hpp`
> 模块: [[crypto|crypto]]

## 概述

SHA-224 哈希工具。提供基于 OpenSSL 的 SHA-224 哈希计算功能，用于 Trojan 协议凭据处理和 HTTP Basic 认证密码哈希。所有函数均为 header-only inline 实现。

## 命名空间

`psm::crypto`

## 依赖关系

| 依赖 | 用途 | 层级 |
|------|------|------|
| `<openssl/sha.h>` | OpenSSL SHA224 API | 外部 |
| `<string>` | `std::string` 返回值 | 标准库 |
| `<string_view>` | `std::string_view` 输入参数 | 标准库 |
| `<array>` | SHA224 摘要缓冲区 | 标准库 |
| `<cctype>` | 十六进制字符判断 | 标准库 |

## 函数

### sha224

功能说明：计算输入字符串的 SHA-224 哈希值，返回 56 字符小写十六进制字符串。

签名：

```cpp
[[nodiscard]] inline auto sha224(std::string_view input) -> std::string;
```

| 参数 | 类型 | 说明 |
|------|------|------|
| `input` | `std::string_view` | 待哈希的输入字符串 |

**返回值**：56 字符的小写十六进制哈希字符串（28 字节二进制摘要转为 56 字符十六进制表示）。

**调用（向下）**：
- `SHA224()` — OpenSSL SHA-224 哈希计算函数
- 逐字节十六进制转换逻辑

**被调用（向上）**：
- `protocol::http::parser` — HTTP Basic 认证中对密码计算 SHA-224 哈希
- `normalize_credential()` — 当输入不是预计算哈希时调用此函数
- `tests/Crypto.cpp` — 单元测试验证
- `tests/E2E.cpp` — 端到端测试凭据生成
- `tests/Http.cpp` — HTTP 测试凭据生成

**涉及的知识域**：
- [[ref/crypto/sha256|SHA-256]] — SHA-2 家族算法原理与对比

### is_hex_string

功能说明：检查字符串是否全部由合法的十六进制字符组成。

签名：

```cpp
[[nodiscard]] inline auto is_hex_string(std::string_view str) -> bool;
```

| 参数 | 类型 | 说明 |
|------|------|------|
| `str` | `std::string_view` | 待检查的输入字符串 |

**返回值**：若字符串只包含 `0-9`、`a-f`、`A-F` 字符则返回 `true`，否则返回 `false`。空字符串返回 `true`。

**调用（向下）**：
- `std::isxdigit()` — 逐字符判断是否为十六进制字符

**被调用（向上）**：
- `normalize_credential()` — 判断凭据是否已是十六进制哈希格式
- `tests/Crypto.cpp` — 单元测试验证

**涉及的知识域**：
- [[ref/crypto/sha256|SHA-256]] — 十六进制表示是哈希输出的通用格式

### normalize_credential

功能说明：将凭据统一为 56 字符 SHA-224 十六进制哈希格式，支持明文密码和预计算哈希两种输入。

签名：

```cpp
[[nodiscard]] inline auto normalize_credential(std::string_view credential) -> std::string;
```

| 参数 | 类型 | 说明 |
|------|------|------|
| `credential` | `std::string_view` | 凭据字符串，可以是明文密码或已计算的 SHA-224 哈希 |

**返回值**：56 字符的小写十六进制 SHA-224 哈希字符串。若输入已是 56 字符的十六进制字符串则直接返回原值，否则计算其 SHA-224 哈希。

**调用（向下）**：
- `is_hex_string()` — 检查输入是否已是十六进制格式
- `sha224()` — 对非哈希输入计算 SHA-224 摘要

**被调用（向上）**：
- `loader::load()` — 配置加载阶段统一用户密码格式，存入账户目录
- `benchmarks/CodecBench.cpp` — 基准测试

**涉及的知识域**：
- [[ref/crypto/sha256|SHA-256]] — SHA-224 与 SHA-256 共享相同的 Merkle-Damgard 结构
- [[ref/protocol/http-connect|HTTP CONNECT]] — HTTP Basic 认证中密码哈希的应用场景
