# x25519.hpp

> 源码: `include/prism/crypto/x25519.hpp`
> 实现: `src/prism/crypto/x25519.cpp`
> 模块: [[crypto|crypto]]

## 概述

X25519 椭圆曲线 Diffie-Hellman 密钥交换。提供 X25519 密钥对生成、公钥推导和共享密钥计算功能。基于 BoringSSL 的底层 `X25519()` / `X25519_public_from_private()` API 实现（比 EVP_PKEY API 快 5-7 倍）。用于 [[stealth/reality|Reality]] 协议的密钥交换和 TLS 1.3 ECDHE。

X25519 使用 Curve25519 椭圆曲线，提供 128 位安全强度，是 TLS 1.3 中 ECDHE 的首选曲线之一。所有密钥和共享密钥长度固定为 32 字节。

同时定义了 Ed25519 密钥对结构体，用于 Reality 协议的服务端自签名证书生成。

## 结构体

### x25519_keypair

X25519 密钥对，包含私钥和对应的公钥，各 32 字节。

```cpp
struct x25519_keypair {
    std::array<std::uint8_t, X25519_KEY_LEN> private_key{};
    std::array<std::uint8_t, X25519_KEY_LEN> public_key{};
};
```

| 字段 | 类型 | 说明 |
|------|------|------|
| `private_key` | `std::array<std::uint8_t, 32>` | X25519 私钥（随机生成的 32 字节标量） |
| `public_key` | `std::array<std::uint8_t, 32>` | X25519 公钥（Curve25519 上的点） |

### ed25519_keypair

Ed25519 密钥对，包含完整私钥（64 字节：种子+公钥）和公钥（32 字节）。

```cpp
struct ed25519_keypair {
    std::array<std::uint8_t, ED25519_PRIVATE_KEY_LEN> private_key{};
    std::array<std::uint8_t, ED25519_KEY_LEN> public_key{};
};
```

| 字段 | 类型 | 说明 |
|------|------|------|
| `private_key` | `std::array<std::uint8_t, 64>` | Ed25519 完整私钥（前 32 字节为种子，后 32 字节为公钥副本） |
| `public_key` | `std::array<std::uint8_t, 32>` | Ed25519 公钥 |

### x25519()

> 源码: `src/prism/crypto/x25519.cpp:41`

执行 X25519 Diffie-Hellman 密钥交换，计算双方共享密钥。

```cpp
auto x25519(std::span<const std::uint8_t> private_key,
            std::span<const std::uint8_t> peer_public_key)
    -> std::pair<fault::code, std::array<std::uint8_t, X25519_SHARED_LEN>>;
```

| 参数 | 类型 | 说明 |
|------|------|------|
| `private_key` | `std::span<const std::uint8_t>` | 本方 32 字节 X25519 私钥 |
| `peer_public_key` | `std::span<const std::uint8_t>` | 对方 32 字节 X25519 公钥 |

- **返回值**: 错误码和 32 字节共享密钥的配对
  - 成功: `{fault::code::success, shared_secret}`
  - 私钥/公钥长度无效: `{fault::code::invalid_argument, 全零}`
  - X25519 计算失败（无效公钥）: `{fault::code::reality_key_exchange_failed, 全零}`
- **调用**:
  - `X25519()` (BoringSSL) — 底层密钥交换，比 EVP_PKEY API 快 5-7 倍
- **被调用**:
  - `reality::auth::authenticate()` — Reality 认证阶段计算共享密钥
  - `reality::handshake::handshake()` — TLS ECDH 阶段计算 TLS 共享密钥
  - `tests/X25519.cpp` — 单元测试验证密钥交换正确性
- **知识域**:
  - [[ref/crypto/x25519|X25519]] — X25519 密钥交换
