# X25519

**类别**: 密码学

## 概述

X25519 是基于 Curve25519 椭圆曲线的 Diffie-Hellman 密钥交换协议，用于在不安全的通道上安全地协商共享密钥。

## 原理

### Curve25519 椭圆曲线

Curve25519 是 Montgomery 形式的椭圆曲线，定义为：
```
y² = x³ + 486662x² + x (mod 2^255 - 19)
```

**特性**：
- 255 位素数域
- 抗时序攻击
- 快速实现

### X25519 密钥交换

**密钥生成**：
1. 生成 32 字节随机私钥
2. 私钥经过位处理（清除最低 3 位，设置最高位）
3. 计算公钥：公钥 = 私钥 × G（基点）

**共享密钥计算**：
1. Alice 发送公钥 A 给 Bob
2. Bob 发送公钥 B 给 Alice
3. 共享密钥 = Alice私钥 × B = Bob私钥 × A

**安全性**：
- 基于椭圆曲线离散对数问题（ECDLP）
- 128 位安全级别
- 抗时序攻击
