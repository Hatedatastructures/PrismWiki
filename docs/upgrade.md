# 升级指南

Prism 版本升级步骤和注意事项。

## 功能简介

提供版本升级、配置迁移、兼容性检查的指导流程。

## 升级步骤

1. **备份配置**: 复制 `configuration.json` 到安全位置
2. **停止服务**: `systemctl stop prism` 或 `sc stop Prism`
3. **下载新版本**: 编译新版本或下载预编译包
4. **更新程序**: 替换旧版本二进制文件
5. **检查兼容性**: 对比新版本配置格式是否有变化
6. **启动服务**: 启动并验证功能

## 配置迁移

新版本可能新增配置项或修改默认值，建议：

- 保留原配置文件核心参数
- 对比新版本默认配置，补充新增字段
- 验证各模块功能正常

## 常见问题

**Q: 升级后配置不兼容怎么办？**
A: 对比新旧配置格式，调整参数名称或结构。

**Q: 升级后连接失败？**
A: 检查协议参数是否变化，确保客户端配置同步更新。

**Q: 如何回滚？**
A: 保留旧版本二进制和配置备份，必要时恢复。
